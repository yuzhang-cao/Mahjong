# 麻将听牌助手（广东/四川，图形化离线 PWA）

## 你提的两点，这个版本都覆盖了
1) **牌是否缺**  
- 四川：万/筒/条（27 张）  
- 广东：万/筒/条 + 字牌（东南西北中发白，34 张）  
- 花牌：春夏秋冬梅兰竹菊（8 张）**可点选记录**，但不计入 13/14 张，也不参与胡牌判定（符合“花牌补牌”的常见处理）。

2) **图形化点牌 + 点击撤销**  
- 先点“万/筒/条/字/花”标签，再点具体牌就会加入手牌  
- 上方手牌区会按“牌名 × 张数”展示  
- 点上方条目即可“减一张”（点错就点掉）  
- 另有“撤销最后一张”和“清空”

## 自动切换规则
- 规则选择为“自动”时：  
  - 手里出现字牌或花牌 -> 自动按广东麻将  
  - 完全没有字/花 -> 自动按四川麻将  
- 你也可手动强制四川/广东；若强制四川但手里有字/花，会提示并阻止计算。

## 计算能力
- 13 张：计算“听哪些牌”
- 14 张：计算“打哪张 -> 听哪些牌”，并判断当前是否已胡
- 四川可选定缺：未清缺按常见规则不听/不胡（并且 14 张时只给“打缺门”的出牌建议）
- 七对：可开关
- 十三幺：仅广东可开关

> 说明：仅做胡牌形判定，不算番，不处理起胡番数（例如“有番才能胡/不能鸡胡”等）。需要的话可继续扩展。

## macOS 本地运行
```bash
cd mahjong-gui-multi-rules-pwa
python3 -m http.server 5173
```

手机与电脑同一 Wi-Fi，手机访问：
`http://<电脑局域网IP>:5173/`

然后“添加到主屏幕”，即可离线使用。
